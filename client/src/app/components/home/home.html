Welcome to our Expense reimbursement System.

problem statement

Current expense reporting is manual, time-consuming, and leads to lost receipts and delayed reimbursements.

solution

Automated workflow from receipt capture to payment with full audit trails and real-time tracking.

project description

ExpenseFlow is an expense report management system that automates the employee reimbursement process. Employees upload receipt images, create expense reports, and submit them for approval. Managers review and approve reports through a workflow system. The application au- tomatically generates formatted Excel reports using Apache POI and processes reimbursements through the Stripe payment API.

technologies used

  • Spring Boot + Spring WebFlux — Reactive REST API backend
  • Angular — Frontend application
  • Hibernate — Database ORM and receipt image storage
  • PostgreSQL — Database
  • Apache POI — Generate formatted Excel reports
  • Stripe API — Payment processing
  • MapStruct — DTO mapping
  • Liquibase — Database versioning and sample data
  • OpenAPI (Swagger) — API documentation
  • Docker — Containerization and deployment
  • Spring Boot Admin — Application monitoring

key features

  • Receipt upload and storage (Hibernate blob storage)
  • Expense entry and categorization
  • Report submission and approval workflow
  • Excel report generation with charts (Apache POI)
  • Payment processing via Stripe API
  • manager approval dashboard.
  • analytics and spending tracking.
